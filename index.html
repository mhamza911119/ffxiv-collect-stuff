Final Fantasy XIV Character Info and Comparison
Welcome to the Final Fantasy XIV Character Info and Comparison Tool. Click on one of the buttons below to select a character whose information to view. Select "Comparison" to compare both characters. Once a character is selected, you can enter a number between 0 and 100 to check the number of achievements that character has obtained that has an own percentage below that number. For example, if you enter 15, it will display the number of achievements the character has obtained that have an ownership rate of less than or equal to 15%. (Try entering larger numbers first, then go lower)
